## Troubleshooting

### "Quiz not found"
- Double-check the Quiz ID spelling
- Ensure the quiz exists in the system
- Contact admin if issue persists

### "No students have attempted"
- Quiz hasn't been taken yet
- Check if quiz is still upcoming
- Verify students have the correct quiz ID

### Charts not loading
- Refresh the page
- Check internet connection
- Try a different browser

### Data seems incomplete
- Ensure students submitted their quizzes
- Check if quiz end time has passed
- Verify backend is running
